Ontology source

Abstract

A base ontology providing a conceptual view into the GeoLink patterns. For data providers, this is the default view of the patterns.

GeologicAgeAppearancec back to ToC or Class ToC

IRI: http://schema.geolink.org/1.0/base/main#GeologicAgeAppearance

This class essentially is intended to capture the relation exemplified by the sentence "Pleistocene has first appearance of 1.6 million years before present". Here, Pleistocene is a geologic age, and 'hast first appearance' points to an entity that has a value component and a unit component.

Data providers can opt not to populate this class directly with explicit URI instances, but rather use blank nodes as instances of this class. An example set of triples would be as follows where the blank node that acts as the filler of the hasFirstAppearance property is an instance of this class:

d:age1 a :GeologicAge .

d:age1 :hasFirstAppearance

[ :hasGeologicAgeValue "1.6"^^xsd:double ;

:hasGeologicAgeUnit iso31:Ma ] .

is in domain of
hasGeologicAgeUnitdp, hasGeologicAgeValuedp
is in range of
hasFirstAppearanceop, hasLastAppearanceop
