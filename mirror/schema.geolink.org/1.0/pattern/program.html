Abstract

Program pattern represents the notion of program as a loose group of things, including cruises, funding awards, activities, events, etc., associated together under one umbrella. Program pattern includes information about its starting and ending dates, agent roles it provides, as well as other information such as webpage, identifiers, names, etc.
